【痰湿体质患者干预方案匹配规律】

1. 中医调理等级完全由初始痰湿积分决定（附表2）：
   - 痰湿积分≤58 → 1级调理（饮食+穴位按摩，30元/月）
   - 59-61 → 2级调理（+八段锦，80元/月）
   - ≥62 → 3级调理（+中药代茶饮，130元/月）

2. 活动干预强度由年龄和活动总分共同约束（附表3）：
   - 年龄越大、活动总分越低，允许的最大强度越低。
   - 实际最优方案中，患者往往选择允许的最高强度以获得最佳降痰湿效果。

3. 每周活动次数选择：
   - 在总成本≤2000元的前提下，优先选择每周5-10次，次数越多效果越好。
   - 成本较高时（如高强度+高次数），可能会被限制。

4. 典型规律示例：
   - 高痰湿积分（≥62）+ 活动耐受好（活动总分≥60）→ 3级中医 + 3级活动强度 + 每周8-10次 → 痰湿积分下降最大。
   - 低痰湿积分（≤58）+ 活动耐受差（活动总分<40）→ 1级中医 + 1级活动强度 + 每周3-5次 → 成本低，渐进改善。
   - 中痰湿积分（59-61）+ 中活动能力（40-59）→ 2级中医 + 2级活动强度 + 每周6-7次 → 平衡成本与效果。
